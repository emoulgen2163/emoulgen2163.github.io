# Emirhan Ülgen — Portfolio Website

A modern, responsive portfolio website showcasing my work as an Android developer and software engineer. Built with clean, semantic HTML, custom CSS, and vanilla JavaScript.

**Live Site:** [emoulgen2163.github.io](https://emoulgen2163.github.io)

---

## 🎯 Overview

This is a full-stack portfolio featuring:
- **About Me** — Professional background, skills, and experience
- **Featured Projects** — Showcasing Android and web development work
- **Contact** — Easy ways to get in touch
- **Responsive Design** — Fully optimized for mobile, tablet, and desktop

---

## ✨ Features

### 🎨 Design & UX
- **Dark Mode Theme** — Modern, comfortable visual aesthetic with custom color palette
- **Smooth Animations** — Fade-in effects on scroll for engaging user experience
- **Glassmorphism Nav** — Blurred background navigation bar
- **Responsive Mobile Drawer** — Hamburger menu with animated drawer for mobile devices
- **Interactive Elements** — Hover effects, smooth scrolling, animated gradient text

### 📱 Mobile First
- Fully responsive layout (mobile, tablet, desktop)
- Touch-friendly navigation drawer for phones
- Optimized viewport settings for all screen sizes

### ⚡ Performance
- Lightweight vanilla JavaScript (no frameworks)
- Custom CSS animations and transitions
- Intersection Observer API for efficient lazy loading of animations
- Minimal external dependencies (only Google Fonts)

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, flexbox, grid, animations, media queries
- **JavaScript (Vanilla)** — Interactive drawer toggle, smooth scroll behavior, intersection observer
- **Google Fonts** — Inter typeface

## 🚀 Getting Started

### View Online
Visit [emoulgen2163.github.io](https://emoulgen2163.github.io) to see the live site.

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/emoulgen2163/emoulgen2163.github.io.git
   cd emoulgen2163.github.io
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python3 -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 🎨 Customization

### Colors
Edit the CSS custom properties in `style.css`:
```css
:root {
  --bg: #0a0a0f;
  --surface: #12121a;
  --accent: #7c3aed;
  --accent2: #06b6d4;
  --text: #e2e8f0;
  --muted: #64748b;
}
```

### Fonts
Update the Google Fonts import or link a different typeface.

### Content
Edit `index.html` to update your information, projects, and experience.
